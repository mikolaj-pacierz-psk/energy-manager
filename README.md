# Energy Manager

## Wymagania

- Python 3.11
- pip 25.0.1

## Uruchomienie aplikacji
- Windows
   ```
   git clone https://github.com/mikolaj-pacierz-psk/energy-manager.git
   cd energy-manager
   python -m venv .venv
   venv\Scripts\activate
   pip install -r requirements.txt
   python main.py
   ```

- Linux
   ```
   git clone https://github.com/mikolaj-pacierz-psk/energy-manager.git
   cd energy-manager
   python -m venv .venv
   source .venv/bin/activate
   pip install -r requirements.txt
   python main.py
   ```
